# 🎬 AI Motivational Content Engine

Fully automated pipeline to generate:

- 🎙️ AI voice (human-like)
- 🧠 Script (DeepSeek via Ollama)
- 🎨 Image (auto-generated)
- 🎬 Video (FFmpeg)
- 📱 Shorts
- 📝 Title, Description, Hashtags

## 🚀 Features

- Offline LLM (DeepSeek)
- Edge-TTS (realistic voice)
- FFmpeg video rendering
- Shorts generation
- Zero-cost pipeline
